# 웹 3D 비행시뮬레이션 — 기능명세서

## 개요
- **타입:** 싱글플레이어 웹 3D 비행시뮬레이션 게임
- **플랫폼:** 데스크톱 웹 (모바일 터치 기본 지원)
- **기술:** Three.js + 단일 HTML (CDN)
- **컨셉:** 오픈 월드 자유비행 — 산, 바다, 도시 위를 자유롭게 비행

## 게임 플레이

### 비행 모델 (Arcade 스타일)
- 피치(상하), 롤(좌우 기울기), 요(좌우 회전)
- 스로틀(가감속) — 최소~최대 속도 제한
- 스톨(stall) — 속도 너무 낮으면 자연 하강
- 부스트 — 순간 가속 (쿨다운 5초)
- 충돌 판정 — 지형/건물 충돌 시 폭발 + 리스폰

### 조작
| 입력 | 키보드 | 효과 |
|------|--------|------|
| 피치 업 | W / ↑ | 기수 올림 |
| 피치 다운 | S / ↓ | 기수 내림 |
| 롤 좌 | A / ← | 왼쪽 기울기 |
| 롤 우 | D / → | 오른쪽 기울기 |
| 스로틀 업 | Shift | 가속 |
| 스로틀 다운 | Ctrl | 감속 |
| 부스트 | Space | 순간 가속 (쿨다운) |
| 카메라 전환 | C | 3인칭 뒤 / 1인칭 콕핏 |
| 일시정지 | ESC / P | 메뉴 |

### 모바일 터치
- 화면 좌측: 가상 조이스틱 (피치/롤)
- 화면 우측: 스로틀 슬라이더
- 부스트 버튼: 우측 하단

## 월드

### 지형
- **Procedural 지형:** SimplexNoise 기반 높이맵
  - 산악 지대 (높은 봉우리)
  - 평원/초원
  - 해안선 + 바다
- **지형 텍스처:** 높이별 색상 그라데이션
  - 해수면: 파랑
  - 저지대: 초록
  - 고지대: 갈색/회색
  - 정상: 하얀 눈
- **크기:** 충분히 넓어서 자유비행 느낌 (10km × 10km 스케일)

### 환경
- **하늘:** 그라데이션 스카이박스 (낮 시간대)
- **구름:** 반투명 빌보드 파티클 (3D 공간에 랜덤 배치)
- **안개:** 원거리 안개 (렌더링 최적화 + 분위기)
- **바다:** 반사 효과 + 약간의 웨이브 셰이더
- **태양:** DirectionalLight + 렌즈플레어(선택)

### 오브젝트
- 체크포인트 링 (통과 시 점수 + 이펙트)
  - 공중에 떠있는 반투명 링
  - 통과 시 색상 변화 + 파티클 폭발
  - 일정 수 통과 → 새 구간 생성
- 도시 건물군 (단순 박스 지오메트리 스카이라인)

## HUD (Head-Up Display)

### 항상 표시
- **속도계:** 현재 속도 (km/h)
- **고도계:** 현재 고도 (m)
- **인공수평의(Attitude Indicator):** 롤/피치 시각화
- **스로틀 바:** 현재 추력 %
- **부스트 쿨다운:** 원형 게이지
- **미니맵:** 좌측 하단 — 현재 위치 + 체크포인트 표시
- **점수:** 통과한 체크포인트 수 / 경과 시간

### 조건부
- **경고:** 저고도, 스톨 임박 시 빨간 경고
- **충돌:** 폭발 이펙트 + "CRASH" 오버레이 + 3초 후 리스폰

## 메뉴

### 시작 화면
- 게임 타이틀 + 비행기 3D 프리뷰
- "비행 시작" 버튼
- 조작법 안내 (접기/펼치기)

### 일시정지 메뉴
- 계속하기
- 조작법
- 재시작

## 성능 최적화
- LOD (Level of Detail) — 원거리 지형 단순화
- 안개로 렌더 거리 제한
- 청크 기반 지형 로딩 (비행기 주변만)
- requestAnimationFrame 기반 렌더 루프
- 그림자 최소화 (태양 그림자 없거나 저해상도)

## 기술 스택
- Three.js (CDN: unpkg 또는 jsdelivr)
- SimplexNoise (인라인 또는 CDN)
- 단일 HTML 파일 (배포 편의)
- GitHub Pages 정적 배포
