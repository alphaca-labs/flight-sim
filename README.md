# SKYFALL — 3D Flight Simulator 🛩️

웹 브라우저에서 즐기는 3D 비행시뮬레이션 게임.

## 🎮 플레이

**[▶ 지금 플레이하기](https://alphaca-labs.github.io/flight-sim/)**

## 조작법

| 키 | 동작 |
|-----|------|
| W / ↑ | 기수 올림 |
| S / ↓ | 기수 내림 |
| A / ← | 왼쪽 롤 |
| D / → | 오른쪽 롤 |
| Shift | 가속 |
| Ctrl | 감속 |
| Space | 부스트 (쿨다운 5초) |
| C | 카메라 전환 (3인칭/1인칭) |
| ESC | 일시정지 |

## 기능

- 🏔️ Procedural 지형 생성 (산, 평원, 바다)
- ☁️ 3D 구름 파티클
- 🎯 체크포인트 링 통과 시스템
- 🏙️ 도시 스카이라인
- 💥 충돌 & 리스폰
- 🎛️ SF 전투기 HUD (속도계, 고도계, 인공수평의, 미니맵)
- 📱 반응형 (데스크톱 최적화)

## 기술 스택

- [Three.js](https://threejs.org/) v0.170 (CDN)
- Simplex Noise (인라인 구현)
- 단일 HTML 파일 (의존성 없음)
